# Code-C-Praktikum-PP
Repository ini berisi kumpulan kode program C++ yang dibuat selama kegiatan Praktikum Pengenalan Pemrograman pada Program Studi Ilmu Komputer, Semester 1, FMIPA Universitas Lampung, tahun 2025.

Kode-kode dalam repository ini digunakan sebagai bahan pembelajaran dan latihan untuk memahami dasar-dasar pemrograman, khususnya menggunakan bahasa C++, sesuai dengan materi yang diberikan selama praktikum berlangsung.
